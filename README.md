# Task2_TypeScript
# Poe analüüsi rakendus

Interaktiivne veebirakendus tooteloendite haldamiseks, mis on kirjutatud TypeScriptis.

## Funktsioonid
- Tooteloendi kuvamine koos hinnangu ja lao staatuse arvutustega.
- Uute toodete lisamine vormi kaudu (omaduste ja mitme lao toega).
- Andmete salvestamine LocalStorage'i (andmed säilivad ka pärast taaskäivitamist).
- Hindade arvutamine allahindlustega kategooria ja minimaalse hinnangu järgi.

## Kuidas projekti käivitada
1. Sõltuvuste installimine:
`npm install`
2. TypeScripti koodi kompileerimine:
`npx tsc`
3. Projekti käivitamine:
- Konsooliaruande vaatamiseks (ülesanne 1): `node dist/task1.js`
- Veebiliidese avamiseks: avage brauseris `index.html` (Live Server).
